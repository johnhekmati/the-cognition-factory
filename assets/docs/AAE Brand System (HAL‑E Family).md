###AAE Brand System (HAL‑E Family)

Version: 1.0
Product: AAE — Adaptive Assessment Engine
Family: HAL‑E (depth engine) · AAE (examination engine)

AAE is the examination layer of the HAL‑E stack — the system that turns mental models into demonstrated mastery.

Family rule:  
HAL‑E teaches. AAE examines.  
Same void, same typographic discipline; different energy metaphor and accent palette.


##1. Brand Essence

Element	                Definition
Name	                AAE — three letters, no hyphen
Expansion	        Adaptive Assessment Engine — use on first mention or in formal headers
One-liner	        Adaptive examination that drives you from knowing to performing.
Promise	                High-fidelity drills, honest signal, path to embodied mastery (retrieval under pressure, not vibes).
Enemy	                Fake confidence; passive “completion”; trivia with no transfer; drilling before a model exists.
Personality	        Examiner + coach — crisp, exacting, fair, never punitive for sport.


Primary line:  
HAL-E teaches. AAE examines.

Supporting lines:

Fidelity when it counts.

Prove the model.

Adaptive pressure, honest signal.

From teach-back to demonstration.


##2. HAL‑E ↔ AAE (Messaging Architecture)

Code
HAL-E (Hyper Accelerated Learning Engine)
   │  depth · maps · spar · teach-back · cartridges
   │
   └── escalates to ──►  AAE (Adaptive Assessment Engine)
                           │  adaptive items · scenarios · retrieval
                           │  error journal · mastery gates
                           └── embodied mastery (perform, not parrot)

                              HAL‑E	                                            AAE
Job	                      Build & stress-test understanding	                    Measure & adapt performance
Metaphor	              Vortex — expansion, synthesis, acceleration	    Aperture — focus, measurement, proof
Palette	                      Blue · cyan · magenta · purple	                    Amber · gold · platinum · ember
Ritual verbs	              MAP, SPAR, LOAD CORE	                            DRILL, GATE, PROVE, REVIEW ERRORS
When first	              Always default	                                    After model exists, or explicit RUN AAE
Failure mode	              Skipping maps	                                    Drilling without architecture


Solo learner copy:  
Use HAL‑E until teach-back is clean; use AAE when you need exam-, cert-, or interview-shaped fidelity — or when retention review says a gap is still lying.


##3. Colorway (AAE)

AAE shares the HAL‑E family base but diverges in accent palette so the two read instantly at a glance.

Shared (Family)
Token	                               Value	                                   Use
Void	                               #000000 – #0A0A0F	                   All hero, slide, video backgrounds
Type primary	                       #FFFFFF (100%)	                           Wordmarks, headlines
Type muted	                       #FFFFFF at 60–75%	                   Subheads, footers
Bridge (optional)	               HAL‑E cyan #00D4FF at low opacity  	   Hairline dividers, “powered by same engine” UI only


AAE Accents (Primary)
Token	Hex	Role
Amber core	#F5B942	Aperture glow, mastery, primary accent
Deep gold	#E8A317	Rings, ticks, secondary glow
Platinum	#C0C8D8	Inner ring, gauge marks, instrument feel
Beam white	#F8FAFC	Center focus, sharp highlights
Ember edge	#FF6B4A	Sparingly — heat, pressure, examination edge


HAL‑E Accents (Reference Only)
Electric blue, cyan, magenta, purple vortex — reserved for HAL‑E teaching moments.

Rules
AAE assets: one focal aperture/gauge; gold + silver; no full-spectrum neon swirl.

Dark-first always; never “white worksheet” as hero.

Ember/orange ≤ 10% of frame.

Cyan bridge only when showing both products on one slide.


##4. Typography

Same discipline as HAL‑E v1.0.

Rituals	                       Monospace or small caps: RUN AAE, GATE, DRILL, ERROR JOURNAL
Pairing	                       HAL‑E left / AAE right, or stacked: HAL-E TEACHES. / AAE EXAMINES.



##5. Imagery & Metaphor

HAL‑E	                                                    AAE
Vortex, schema graph nodes, convergence	                    Aperture, reticle ticks, gauge arc, focused beam
Many connections	                                    Single question: can you demonstrate it?
Particle drift (exploration)	                            Particle snap (alignment, lock-on)


Use: rings, radial ticks, narrow cones of light, crystalline sharpness.
Avoid: graduation caps, red X failure theater, gamified streak flames, quiz-app UI.

Embodied mastery: scenarios, timed retrieval, explain-under-constraint — not only MCQ bubbles.


##6. Voice & Tone

Dimension	AAE Guideline
Register	Clinical warmth — exact, fair, adaptive
Challenge	“Show me.” “Again, without notes.” “Your error pattern is X.”
Avoid	        Shame, hype, guaranteed pass
Celebrate	Mastery gates cleared, error journal shrinking, transfer


Say: adaptive, fidelity, retrieval, gate, error journal, scenario, demonstrate, embodied mastery.
Avoid: “easy quiz,” “AI graded instantly,” replacing HAL‑E depth.


##7. Logo / Lockup Usage

Primary: AAE wordmark + expansion + amber aperture on void.

Relationship lockup: Dual headline slide:
HAL-E TEACHES. AAE EXAMINES.

Do
Keep reticle/aperture centered and symmetrical

Gold + platinum as dominant accents

Clear space ≥ height of “A”

Show HAL‑E → AAE escalation in diagrams

Don’t
Reuse HAL‑E vortex as AAE hero

Lead with blue-purple on AAE-only pages

Use busy backgrounds without scrim

Imply AAE is optional fluff
